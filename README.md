
# 📚 UUKS 小说下载器 & 电子书生成器 - 用户手册

欢迎使用本工具！这是一款**全自动**的小说下载软件，专门用于将在线小说保存为电脑上的文本文件（TXT），并一键制作成精美的电子书（EPUB），方便您导入到 iPhone (iBooks)、Kindle 或其他阅读器中阅读。

现有版本只针对 https://www.uuks.org/ 

安装直接使用install_all.bat
第一次会生成start.bat

启动爬虫使用：start.bat
---

## ⚠️【使用前必读：网络环境】

**本软件所采集的目标网站（UUKS）位于海外，国内网络无法直接访问。**

在使用本软件前，请务必确认：
1.  **您已开启 代理软件 (VPN/加速器)**。
2.  **无需在代码里设置代理**：本软件会自动检测并读取您电脑系统的代理设置（System Proxy）。
    *   *简单来说：只要您的浏览器（如 Chrome/Edge）能正常打开该小说网站，本软件就能正常工作。*

## 🚀 快速开始 (3步搞定)

### 第一步：启动软件
1. 找到文件夹中的 **`start.bat`** 文件。
2. **双击运行**它。
3. 会出现一个黑色的命令行窗口，这就是软件的操作界面。

### 第二步：输入小说链接
1. 在浏览器中打开您想下载的小说目录页（例如：`https://www.uuks.org/b/12345/`）。
2. 复制这个网址。
3. 在软件窗口中，**粘贴网址**，然后按 **回车键 (Enter)**。

### 第三步：按顺序操作
软件菜单就像流水线一样，请按照 **1 -> 2 -> 4** 的顺序操作：

#### 1️⃣ 输入 `1` 并回车：【抓取目录】
*   **作用**：软件会启动一个浏览器窗口，自动去网站上把所有章节的标题记录下来。
*   **注意**：如果看到浏览器显示“正在验证 (Just a moment...)”，**请不要关闭它**，软件会自动处理。
*   **完成标志**：看到提示 `[完成] 目录文件已生成`。

#### 2️⃣ 输入 `2` 并回车：【批量下载】
*   **作用**：软件开始一章一章地把正文下载到电脑里。
*   **状态**：屏幕上会显示 `[1/1000] 下载: 第一章...`。
*   **提示**：如果不小心关闭了窗口，下次运行选 `2`，软件会自动**跳过**已下载的章节，实现断点续传。

#### 4️⃣ 输入 `4` 并回车：【制作电子书】
*   **作用**：将下载好的几百个 TXT 文件，配合**元数据**，打包成一本精美的 EPUB 电子书。
*   **确认信息**：屏幕上会显示即将生成的书本信息。如果不满意，您可以根据提示输入 `m` 进行修改。

## 📖 关于书籍信息与封面 (元数据)

本软件不仅仅是下载文字，它还会自动帮您完善书籍的**元数据 (Metadata)**，让您的电子书在阅读器里漂漂亮亮。

### 1. 自动获取的信息包括：
*   **书籍封面 (Cover)**：软件会自动下载小说网站上的高清封面图。
*   **书籍简介 (Intro)**：自动抓取小说的内容简介，嵌入电子书的扉页。
*   **作者与书名**：自动识别并填充。

### 2. 如何确认与修改？
当您执行 **功能 `4` [制作电子书]** 时，软件会先在屏幕上列出它获取到的信息：
> ```text
> 书名: xxxx
> 作者: xxxx
> 简介: (已获取)
> 封面: cover.jpg (存在)
> ```
*   **如果您觉得信息准确**：直接按 **回车**，立即生成。
*   **如果您想修改 (例如书名有乱码)**：输入 **`m`** 并回车，按照提示手动输入新的书名或作者名。

### 3. 如何自定义封面？(高级玩法)
如果您不喜欢网站自带的封面，或者软件没下载到封面：
1. 打开 `novels` -> `[小说名]` 文件夹。
2. 找一张您喜欢的图片（jpg格式）。
3. 将其重命名为 **`cover.jpg`**。
4. 放入该文件夹（如果已有同名文件，直接覆盖）。
5. 重新运行 **功能 `4`**，生成的电子书就会使用您的新封面了！

## 🛠️ 常见问题解答 (Q&A)

### Q1: 软件一打开就报错，或者浏览器一片空白？
**答**：请检查您的 VPN 是否开启了**全局模式**。如果浏览器都打不开小说网，软件也无法工作。

### Q2: 生成的电子书缺章节，比如一共 100 章，只有 17 章？
**答**：这通常是因为**下载没完成**（Step 2）就开始打包了。请重新运行功能 **`2`**，等它跑完所有章节，再运行功能 **`4`**。

### Q3: 为什么会弹出一个浏览器窗口？我可以关闭它吗？
**答**：软件需要模拟真人浏览网页来绕过网站的反爬虫验证。任务运行期间，**请不要手动关闭该浏览器窗口**，最好把它最小化。

### Q4: 下载的小说文件在哪里？
**答**：在软件同级目录下，有一个 **`novels`** 文件夹。点进去，以小说书名命名的文件夹里，存放着所有源文件和最终生成的 `.epub` 电子书。
